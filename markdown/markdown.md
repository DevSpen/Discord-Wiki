# Markdown
Discord has its own version of [markdown](https://en.wikipedia.org/wiki/Markdown). It allows you to apply awesome text effects in your message!

### Key
- 1: \*\*bold\*\*
- 2: \*italics\*
- 3: \_\_underline\_\_
- 4: \`code-snippet\`
- 5: ||spoiler||
- 6: \~\~strike-through\~\~
- 7:

\> quote
- 8:
```
>>> multi
line
quote
```
